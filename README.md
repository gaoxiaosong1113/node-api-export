
<h1 align="center">node-api-export</h1>
<div align="center">
一个快速生成api请求的工具
</div>

## ⌨️ 目的 

解决前端项目中需要从api文档手动复制api接口道项目里的问题，可通过本工具一次性生成所需的api请求，文档更新后，重新生成即可。


## 📦 安装

```bash
npm install
```

## 🔨 示例

```bash
# 输入地址时只需输入前缀即可
# 如文档地址为 http://39.106.180.25:8091/swagger-ui.html 只需入http://39.106.180.25:8091
npm run api
```


## 🔗 支持列表

- swagger
- doclever
